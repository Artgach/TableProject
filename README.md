# Таблица пользователей (тестовое задание)

Приложение показывает список пользователей из API https://dummyjson.com/users с возможностью:
- сортировки (по ФИО, возрасту, полу, телефону)
- фильтрации (поиск по имени/email/телефону, пол, точный возраст)
- пагинации
- изменения ширины колонок
- просмотра детальной информации в модальном окне

## Стек
- React 18
- Vite
- Bootstrap 5 + react-bootstrap
- Fetch API
- react-resizable (для изменения ширины колонок)

## Функционал
- Таблица: фамилия, имя, отчество, возраст, пол, телефон, email, страна, город
- Сортировка: asc / desc / без сортировки (клик по заголовку)
- Фильтры: комбинированные (поиск + пол + возраст)
- Пагинация: 150 записей на страницу (для точной фильтрации)
- Модальное окно: ФИО, возраст, адрес, рост, вес, телефон, email, аватар
- Обработка ошибок и загрузки
- Адаптивность: max-width 1400px
